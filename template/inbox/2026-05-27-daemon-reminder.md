The daemon handles background tasks like scheduled routines and memory synthesis. Without it, you can still run skills manually in Claude Code, but nothing will run on a schedule.

To install, paste this into Claude Code:

> Help me install the differnet daemon so my routines can run automatically.

Once running, the Routines page will show a green status dot.
